## ✨ Features

- **Modern Sidebar Navigation** – Icons with hover tooltips, smooth scrolling to sections.
- **Dynamic Banner with Carousel** – Two background images slide automatically (developer workspace, code editor).
- **Typing Animation** – Banner text changes dynamically (UI/UX Designer, Front End Developer, Backend Developer).
- **About Section** – Profile image, personal details, social links, and an expandable info card.
- **Education & Experience** – Timeline‑style layout with colour‑coded years and descriptions.
- **Progress Bars** – Circular animated counters for projects, team work, happy clients, creativity.
- **Services & Tech Stack** – Icons and descriptions for HTML5, CSS3, Bootstrap, JavaScript, PHP, jQuery, JSON, MySQL.
- **Portfolio Gallery** – Filterable grid (Design, Development, Photoshop, Template) with lightbox popup.
- **Testimonial Slider** – Client feedback with video play button (YouTube embed).
- **Blog Section** – Recent news/articles with “Read More” links to `blog.html`.
- **Interactive World Map** – Vector map with markers (jVectorMap).
- **Contact Form** – PHP backend with AJAX submission, JSON response handling.
- **Footer** – Location, contact numbers, email addresses, and copyright.
- **Go‑to‑Top Button** – Smooth scroll to top.
- **Preloader** – Loading animation before page content appears.
- **Fully Responsive** – Custom media queries for all screen sizes.
